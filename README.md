## How to Use

1. **Open the Website**: Simply open `index.html` in any modern web browser
2. **Contact Form**: Fill out the contact form and it will open your email client with a pre-filled message
3. **Navigation**: Use the header navigation or scroll to explore different sections
4. **Mobile**: The site automatically adapts to mobile devices with a hamburger menu

## Customization

### Colors
The main color scheme can be changed in `styles.css`:
- Primary color (yellow/gold): `#ffc107`
- Dark background: `#1a1a1a`
- Text color: `#333`

### Contact Information

- Email: `riad.nakhoul1@outlook.com`
- Phone: `+1-226-280-0591`
- Service areas and hours

### Form Email
The contact form sends to: `riad.nakhoul1@outlook.com`
